FOR TESTING PURPOSES ONLY

Note that this is not comprehensive demo of DOMani's capabilities. I designed this page to test DOMani as I wrote it. That being said, please feel free to experiment with it as well following the instructions below.

  1. Open Chrome DevTools using Cmd + Opt + I on Mac or Ctrl + Shift + I on Windows.
  2. Examine the diffrent HTML elements given.
  3. Test DOMani's core function, $l, and its DOM manipulation and traversal methods on the elements using the console.
    For example, enter $l('.greetings-container').html('<p>One greeting to rule them all</p>') to test #html.
    See the README for a complete list of methods.
